# Learning Management System (Backend)

## Overview
This project is a backend implementation of a Learning Management System developed during an internship. It focuses on user authentication, role-based access, and RESTful API development to support core LMS functionalities.

## Key Features
- User authentication and authorization
- Role-based access control for different user types
- Student and course data management
- REST APIs for backend operations

## Tech Stack
- Python
- Django
- Django REST Framework
- MySQL

## Responsibilities
- Designed database models to manage users and learning data
- Built REST APIs for authentication and core LMS operations
- Implemented secure access control for different user roles

## How to Run
1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Configure database settings
4. Run the Django server
